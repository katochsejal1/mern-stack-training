USING SEMANTIC TAGS (TAGS THAT HAVE MEANINGS)

  1. HEADER TAG

    THIS IS HEADER TAG

    THE HEADER TAG IS USED TO DEFINE THE HEADER SECTION OF THE HTML DOCUMENT. IT USUALLY CONTAINS INTRODUCTORY CONTENT OR NAVIGATION LINKS OR LOGOS.

  2. MAIN TAG

    THIS IS MAIN TAG

    THE MAIN TAG IS USED TO DEFINE THE MAIN CONTENT OF THE HTML DOCUMENT. IT USUALLY CONTAINS THE PRIMARY INFORMATION THAT IS RELEVANT TO THE TOPIC OF THE PAGE.

  3. ARTICLE TAG

    THIS IS ARTICLE TAG

    THE ARTICLE TAG IS USED TO DEFINE A SELF-CONTAINED PIECE OF CONTENT THAT CAN BE INDEPENDENTLY DISTRIBUTED OR REUSED. IT USUALLY CONTAINS A COMPLETE THOUGHT OR TOPIC, SUCH AS A BLOG POST, NEWS ARTICLE, OR FORUM POST.

  4. SECTION TAG

    THIS IS SECTION TAG

    THE SECTION TAG IS USED TO DEFINE A DISTINCT SECTION OF A DOCUMENT OR WEBPAGE. IT IS USUALLY CONTAINS A GROUP OF RELATED CONTENT THAT CAN BE SEPARATED FROM THE REST OF THE DOCUMENT.

  5. ASIDE TAG

    THIS IS ASIDE TAG

    THE ASIDE TAG IS USED TO DEFINE CONTENT THAT IS RELATED TO THE MAIN CONTENT OF THE PAGE BUT IS NOT ESSENTIAL TO ITS UNDERSTANDING. IT USUALLY CONTAINS ADDITIONAL INFORMATION, SUCH AS SIDEBARS, QUOTES, OR ADVERTISEMENTS.
